# Agent Runtime Workflow Assets

- workflow definitions,
- constitutional rule assets,
- prompt templates tied to workflow stages,
- human review and approval gate specifications,
- portable agent adapter notes,
- example runtime configuration notes for OpenClaw and Hermes,
